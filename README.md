## Getting Started

### Prerequisites

- Node.js 22.x
- PNPM 10.x

### Local Setup

1. Check if you have corepack:
   ```sh
   corepack -v
   ```
1. If not, install it:
   ```sh
   brew install corepack
   ```
1. Enable pnpm
   ```sh
   corepack enable pnpm
   ```
1. Install pnpm
   ```sh
   corepack use pnpm
   ```
1. Install project dependencies
   ```sh
   pnpm install
   ```
1. Start the service:
   ```sh
   pnpm start
   ```
